# Crypto Dashboard

Full-stack crypto tracking app with real-time prices, 
interactive charts and JWT authentication.
Built with React + TypeScript on the frontend, Node.js + SQLite on the backend

## Preview

![Dashboard main view](./public/preview.png)
![Price chart view](./public/preview2.png)

## Features
- Live prices for 50+ cryptocurrencies
- Interactive price charts with timeframes (1D/1W/1M)
- Favorites system
- JWT Authentication

## Tech Stack
**Frontend:** React, TypeScript, Zustand, lightweight-charts, shadcn/ui, Tailwind CSS
**Backend:** Node.js, Express, JWT, bcrypt, SQLite

## Live Demo
- Frontend: https://crypto-dashboard-jet.vercel.app/

## Run locally
```bash
# Frontend
cd crypto-dashboard
npm install
npm run dev
```
```bash
# Backend
cd crypto-dashboard/server
npm install
npm run dev
```
